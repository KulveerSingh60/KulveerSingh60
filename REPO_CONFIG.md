## Repository Descriptions

### Portfolio Repository
**Description:** IDE/Terminal-inspired personal portfolio website showcasing full-stack development skills, projects, and experience. Built with HTML5, CSS3, JavaScript, and Bootstrap 5.

### Inventory System Repository
**Description:** Full-stack PHP and MySQL inventory management system with user authentication, product tracking, sales/purchase management, and analytics dashboard. Production-ready application for business inventory control.

### Projects Repository
**Description:** Comprehensive showcase of live web applications and portfolio projects with detailed documentation, featuring real estate platform and inventory management system.

## How to Update Topics on GitHub

1. Go to your repository
2. Click **Settings** (gear icon)
3. Scroll to **Topics** section
4. Add the topics listed above
5. Click outside the field to save
